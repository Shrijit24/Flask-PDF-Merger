# PDF Merger Web Application

## Introduction
In today’s digital world, managing multiple PDF files efficiently is a common need. This project is a **web-based PDF Merger** built using **Python** and **Flask**, allowing users to merge multiple PDF files into a single PDF quickly and easily. The application is lightweight, user-friendly, and works entirely in a web browser, without requiring any additional software.

## Objectives
- Provide an easy-to-use web application to merge multiple PDF files.  
- Allow users to upload any number of PDF files and merge them in the desired order.  
- Demonstrate the use of **Python’s PyPDF2 library** for PDF manipulation.  
- Build a simple interface using **Flask** and **HTML**.  
- Enable instant download of the merged PDF file.  

## How the Application Works
1. Open the application in a browser.  
2. On the home page, enter the number of PDF files you want to merge.  
3. Upload the files in the order you want them merged.  
4. Click the **Merge PDFs** button.  
5. The application uses **PyPDF2** to combine the files in the specified order.  
6. Download the merged PDF instantly.  

## Technologies Used
- **Python** – backend logic and PDF processing  
- **Flask** – web application framework for routing and handling requests  
- **PyPDF2** – library for reading, merging, and writing PDF files  
- **HTML** – user interface
